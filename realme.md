# 🚀 Cyber Login Calculator

Projeto front-end com sistema de autenticação simulada + calculadora moderna com interface interativa em Canvas.

🔗 Deploy: https://cyber-calculatora.vercel.app/

---

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com foco em:

- Simulação de autenticação (registro e login)
- Proteção de rotas no front-end
- Manipulação de estado com localStorage
- Interface moderna com partículas animadas
- Experiência visual interativa
- Responsividade para desktop e mobile

A proposta foi criar algo visualmente impactante para portfólio, mas mantendo lógica real de navegação e controle de acesso.

---

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Canvas API
- LocalStorage
- Deploy na Vercel

---

## 🔐 Funcionalidades

✅ Cadastro de usuário  
✅ Login com validação  
✅ Proteção de rota (acesso apenas logado)  
✅ Logout funcional  
✅ Fundo animado com partículas interativas  
✅ Calculadora estilizada no mesmo padrão visual  
✅ Layout responsivo
